1) definir les plugins utilisés dans les subprojects:

Dans FanControl/build.gradle:

buildscript {
    repositories {
        gradlePluginPortal()
    }
    dependencies {
        classpath 'com.jfrog.bintray.gradle:gradle-bintray-plugin:1.8.5'
    }
}


On peut ensuite utiliser ce plugin dans un subproject avec:


apply plugin: 'com.jfrog.bintray'




settings.gradle:
on défini le rootProject.name:
rootProject.name = 'FanControl'
on défini les subProjects:
include 'proto'
include 'app'



repository: propre a chaque module.
On en aura besoin dans le FanControl build.gradle pour recuperer les plugins.
gradlePluginPortal()

et dans proto and app build.gradle.
